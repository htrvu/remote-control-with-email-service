Since you choose Run in Home page, your computer can be controlled by someone that is allowed by you.

How can they do that? With our host mail g8rc.host@gmail.com, they will send mails to this email address in a specific format for header: Starting with [G8RC] and followed by a command code.

What things can they do with your PC? Get the MAC address of your computer, some operations with apps and processes, screen and webcam data, keylogger, modify registry value, file manipulations,... For more detail, the controllers can see all of commands by sending an email with header [G8RC] HELP to the host mail.

Also, there are 2 type of controller:

  • Basic controller: They can control something related to MAC address, App, Process, Screen, Webcam and Keylogger

  • Advanced controller: They can do everything our application produces, including basic commands and Explorer, Registry and PC shutdown/restart. .

Notice: Before running this app, please go to Configurations to set up everything needed. If you want to change to configurations while this app is running, you can open this window from the tray icon on the taskbar. And finally, you can choose Exit to stop the effects of this application.
